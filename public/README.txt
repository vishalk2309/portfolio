Put your own photo here.

1. Save your photo into this `public` folder named:  profile.jpg
   (a .png or .webp is fine too — just match the name in src/data.js)

2. It is already referenced in src/data.js as:  profile.photo: "/profile.jpg"

Anything in this `public` folder is served at the site root, so
public/profile.jpg  ->  /profile.jpg
